实验3

实验题目：
利用云平台搭建个人博客

实验目的：
本实验旨在通过实践操作，掌握基于云计算环境的网站部署技能。

具体包括：
构建环境并实现个人博客/网站的搭建，撰写实验报告，并将这份报告作为一篇博文发布在自己的博客 / 网站上。

硬件环境： 
联网的计算机一台

软件环境：
Windows or Linux

实验步骤与内容：
GitHub Pages + Hexo搭建个人博客网站

gitbub准备工作：s
创建仓库


格式必须是：用户名+.gitee.io 只有这样，将来要部署到Gite e page的时候，才会被识别，也就是xxxx.gitee.io，其中xxx就是你注册Gitee的用户名

hexo准备工作
安装

初始化hexo，新建存储博客的文件夹

安装npm

进入文件夹，可以看到hexo站点已经安装完成

安装一个hexo的上传插件deploy-git

启动hexo站点
hexo g
hexo server

访问http://localhost:4000/

将hexo博客站点上传到github上
修改hexo配置文件_config.yml，指定仓库路径

链接github
git config –global user.name “xinyi-0509”
git config –global user.email “f2803602080@gmail.com“

创建 SSH 密匙：
ssh-keygen -t rsa -C “GitHub 邮箱”

添加密匙：
clip < C:\Users\Administrator.ssh\id_rsa.pub
将公钥复制到剪贴板。
打开 GitHub SSH Keys 页面
点击 New SSH Key，粘贴密钥，点击 Add SSH Key

测试 SSH 连接：
ssh -T git@github.com

上传网站到github pages
